# 🦞 OpenClaw 联网搜索哪家强

> 来源：小红书笔记「🦞 Openclaw 联网搜索哪家强」
> 发布者：小红书作者（clawdbot 用户）

## 背景

作者一开始用 OpenClaw/Clawdbot 时用的 Opus 4.5，自带的联网功能非常丝滑，但钱包扛不住。

切到官方推荐的 Minimax 后发现最大的问题：**不自带搜索**。官方推荐的 Brave 搜索还要绑卡，比较麻烦。

于是作者让 clawdbot 做了几个 skill 进行对比测试。

## 测试方案

为了公平，作者让 agent 想了 **10 个测试用例**，并生成了一份对比报告（笔记中附有图表和对比截图）。

## 四种搜索方案详解

### 1️⃣ Serper（Google 搜索）

**本质：** 其实就是 Google 搜索的 API 封装。

**优点：**
- 外文搜索又新又全
- 有 **2500 次免费额度**，最关键的是**不用绑卡**
- 覆盖面广，连 X/Twitter 内容都能搜

**白嫖技巧：** 用完 2500 次之后换个邮箱注册，继续免费续杯。

**配置 Prompt：**
> 帮我搭一个 serper dev 的搜索 skill，需要 api key 搭的时候告诉我

**注册地址：** https://serper.dev

### 2️⃣ Perplexity

**地位：** AI 搜索的代表产品。

**作者获赠方式：** 因为买了 Lenny Bundle 有个 Pro 送了额度，所以配上了做对比。

**评价：** 又快又稳，**体验最好的方案**。

### 3️⃣ Bird（X/Twitter 搜索）

**来源：** OpenClaw 自带的小龙虾搜索工具。

**用途：** 专门用来搜索 X（Twitter）。

**配置：** 需要按指南配置 cookies。

**评价：** X 上的定向资讯搜索，**非常稳**。

### 4️⃣ Minimax（中文搜索 skill）

**背景：** Minimax 其实有 search tool 功能，但**没有集成到模型里**。作者是单独让它封装了一个 skill 出来用的。

**优点：**
- 中文搜索效果好
- 能覆盖 B 站内容

**缺点：**
- **速度慢**（最大缺点，没有之一）

**配置方式：**
> 类似的，互动口喷就可以配置完

也就是说告诉 agent "帮我搭一个 minimax 搜索 skill"，它会引导你完成配置。

## 测试报告结论

| 排名 | 方案 | 推荐场景 | 一句话总结 |
|------|------|---------|-----------|
| 🥇 | **Perplexity** | 钱包 ok 的情况 | 又快又稳，体验最好 |
| 🥈 | **Serper** | 白嫖党 | 免费额度大，覆盖面广，连 X 都能搜，用完换邮箱继续免费续杯 |
| 🥉 | **Bird** | X 定向资讯搜索 | 很稳，必备 |
| 4 | **Minimax** | 中文 / B 站搜索 | 可以覆盖 B 站，最大缺点就是慢 |

## 最终建议

> **把这些策略告诉小龙虾，让它记住并按需调度。**

也就是说配置好多个搜索 skill 后，不是固定用某一个，而是让 agent 根据搜索场景**智能选择最合适的工具**——搜中文用 Minimax、搜外文用 Serper、搜 X 用 Bird，钱包管够用 Perplexity。

## 当前本机配置

当前本机 OpenClaw 使用的搜索引擎为 **Tavily**（默认）：
```json
{
  "tools": { "web": { "search": { "provider": "tavily", "enabled": true } } },
  "plugins": {
    "entries": {
      "tavily": {
        "enabled": true,
        "config": {
          "webSearch": {
            "apiKey": "tvly-dev-..."
          }
        }
      }
    }
  }
}
```
