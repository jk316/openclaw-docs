# 飞书集成

飞书作为主要消息通道。

## 文档操作

- 通过飞书开放 API 直接操作（RESTful）
- 使用应用 token 认证（`tenant_access_token`）
- 支持创建文档、写入表格、删除表格等操作
- 表格 API 有限制：一次性创建大尺寸表格可能失败

## 权限说明

| 权限 | 功能 | 状态 |
|------|------|------|
| `docx:document` | 创建/读写文档 | ✅ 已有 |
| `drive:drive` | 上传文件到云盘 | ❌ 需要额外申请 |

## 踩坑记录 🕳️

### API 调用
- curl 比 Python requests 在飞书 API 调用中更稳定（避免 JSON 编码问题）
- 文档操作失败后需重新创建文档（旧文档结构可能损坏）
- Excel 区域操作 API 有行数限制

### 表格操作
- 单次创建大尺寸表格可能失败（行数限制）
- 建议分步操作：先创建小表格，再逐行写入
- cell_id 404 错误通常是因为文档结构已被破坏，需要重建文档
