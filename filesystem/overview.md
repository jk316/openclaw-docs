# 文件系统

## 工作目录

- **主工作目录**：`~/.openclaw/workspace/` — 所有文件操作都在此目录
- **可达目录范围**：以当前用户身份可访问的所有路径
- 不会主动碰系统关键目录或用户隐私文件

## 记忆系统

- **长期记忆**：`MEMORY.md` — 跨会话持久化，记录重要决策和上下文
- **每日日志**：`memory/YYYY-MM-DD.md` — 每日活动记录

## 配置文件

- **主配置**：`~/.openclaw/openclaw.json`
- **技能文档**：`SKILL.md` — 每个技能的说明

## 访问边界

| 类型 | 可操作 |
|------|--------|
| workspace 目录 | ✅ 完全 |
| OpenClaw 配置目录 | ✅ 读写 |
| /tmp 临时目录 | ✅ 完全 |
| 个人文件夹 | ✅ 但需明确要求 |
| 系统关键目录（/etc 等） | ❌ 不主动碰 |
| sudo 权限 | ❌ 需用户授权 |
