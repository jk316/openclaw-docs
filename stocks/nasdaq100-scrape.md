# NASDAQ 100 历史数据爬取

## 背景

爬取 NASDAQ 100 全部 101 只股票（含 GOOG/GOOGL 两只 Alphabet 股票）的历史日线数据。

## 结果

| 指标 | 数值 |
|------|------|
| 股票数 | 101 只 |
| 总记录数 | 749,421 条 |
| 成功 | 101/101 |
| 失败 | 0 |
| 耗时 | 247 秒（4.1 分钟） |

## 数据格式

每只股票输出一个 JSON 文件：
```json
{
  "ticker": "AAPL",
  "company": "Apple Inc.",
  "latest_price": 310.44,
  "records": 11454,
  "start_date": "1980-12-12",
  "end_date": "2026-05-26",
  "data": [
    {"date": "1980-12-12", "open": 0.10, "high": 0.10, "low": 0.10, "close": 0.10, "volume": 104437200}
  ]
}
```

## 存储

| 文件 | 大小 |
|------|------|
| JSON 文件（101 个） | ~156MB |
| 汇总 CSV | 50MB |
| 摘要 JSON | 19KB |

## 技术栈

- **工具**：yfinance（Python 库）
- **数据范围**：从各股票 IPO 到最新（最早 1980 年）
- **API 调用**：每只股票 1 次 yfinance 请求，爬5年和爬全部请求次数相同
