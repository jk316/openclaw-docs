# 股票数据获取

## 数据源对比

| 数据源 | 价格 | 支持范围 | 限制 |
|--------|------|---------|------|
| **yfinance** | 免费无限制 | 全球股票全部历史日线数据（1980至今） | 无 |
| **Finnhub** | 免费受限 | 实时报价、搜索、公司信息 | K线需要付费套餐 |

### yfinance

- 完全免费无限制
- 支持全部历史日线数据（从 IPO 到最新）
- 数据维度：日期、开盘(O)、最高(H)、最低(L)、收盘(C)、成交量(V)
- 示例：AAPL 从 1980-12-12 至今共 11,454 条记录

### Finnhub

免费 API key 支持：
- `/quote`（实时报价）
- `/search`（搜索）
- `/stock/symbol`（列表）
- `/stock/profile2`（公司信息）
- `/news`（新闻）

收费/高级套餐才支持：
- `/stock/candle`（OHLCV K线）— 实测 403
- `/stock/tick`（逐笔成交）
- `/stock/profile`（详细公司信息）

## 发送方式

通过 163 邮箱 SMTP 发送附件到 QQ 邮箱：
- 需要 163 邮箱的授权码（非登录密码）
- 授权码可能过期，需定期更新
